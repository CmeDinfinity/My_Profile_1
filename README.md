 # Personal Profile Page

## About

This is a simple Personal Profile Page created using HTML. The project demonstrates the use of semantic HTML elements and proper webpage structure.

## Features

* Header section with name and role
* About Me section
* Skills section with an unordered list
* Contact section with email, phone number, and LinkedIn link
* Semantic HTML structure using `<header>`, `<main>`, and `<section>`

## Technologies Used

* HTML5

## Purpose

The purpose of this project is to practice semantic HTML and create a well-structured personal profile webpage.
